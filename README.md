# AI主导的管理工具

一张画布管理万物 | 分支对话系统

## 功能特性

### 🎨 一张画布管理万物
- 交互式画布界面，支持网格背景
- 多种工具：矩形、圆形、文本、便签
- 可视化项目管理
- 拖拽操作支持

### 💬 分支对话系统
- AI智能对话助手
- 支持从任意对话点创建分支
- 基于关键词的智能回复
- 分层对话显示

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
# 同时启动前端和后端
npm run dev

# 或者分别启动
npm start          # React 前端 (端口 3000)
npm run server     # Express 后端 (端口 3001)
```

### 访问应用
打开浏览器访问 http://localhost:3000

## 使用指南

### 画布操作
1. 使用工具栏添加不同类型的元素
2. 点击元素进行选择和编辑
3. 拖拽移动元素位置
4. 使用"清空"按钮重置画布

### 对话系统
1. 在输入框中输入问题或想法
2. AI助手会智能回复相关建议
3. 点击"创建分支"开始新的讨论线程
4. 支持的关键词：任务、项目、画布、分支、帮助

## 技术栈
- **前端：** React 18 + Fabric.js
- **后端：** Express.js + Node.js
- **样式：** 原生 CSS
- **AI集成：** 关键词匹配系统
